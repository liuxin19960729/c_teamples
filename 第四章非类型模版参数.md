## 4.1 非类型的类模版参数
[code](./codes/chapter4/Stack.hpp)
```cpp
template <typename T, int MAX_SIZE>
class Stack
{
private:
    T elems[MAX_SIZE]; //包含元素的数组
    size_t nums = 0;

public:
    Stack(/* args */);
    ~Stack();
    void push(T const &);
    T pop();
    T top() const;
    bool empty()
    {
        return MAX_SIZE == 0;
    }
    bool full() const
    {
        return nums == MAX_SIZE;
    }
};


template <typename T, int MAX_SIZE>
MAX_SIZE 非类型模版参数


template <typename T, int MAX_SIZ=50>

为模版非类型参数指定默认值

note:
Stack<int,10> 和  Stack<int,20> 他们并不是同一个类型
```
